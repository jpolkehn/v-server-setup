## Installation

### Voraussetzungen
- Node.js 14.x
- NPM 6.x

### Installation
1. Clone the repository
2. Install the dependencies
```bash
npm install
```
3. Start the server
```bash
npm start
```

## Usage

### Start the server
```bash
npm start
```

### Stop the server
```bash
npm stop
```

### Restart the server
```bash
npm restart
```
